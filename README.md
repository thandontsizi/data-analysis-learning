# Data Analysis Learning:
This repository contains notes, practice, tests, and exploratory work used while learning data analysis. 
It focuses on building strong fundamentals, practicing core tools, and applying them to realistic analysis problems.

The goal of this portfolio is not just to show results but to show how I think, how I learn, and how I approach data problems end-to-end.

## Repository Structure:
The repository is organised into learning sections. 
Each section contains notes, commands, and self-tests.
Key sections are:
- 00-PostgreSQL-Fundamentals/
- 01-SQL-Analysis/
- 02-Python-for-Data-Analysis/
- 03-Exploratory-Data-Analysis/
- 04-Data-Visualisation/

## Most Learning Sections Contain a:
- README.md: concepts, objectives, and explanations.
- commands.md: tools, syntax, and commonly used commands.
- tests.md: self-assessment questions.

## Tools and Technologies:
The main tools used across this portfolio include:
- SQL(PostgreSQL).
- Python (pandas, NumPy, matplotlib, seaborn).
- Jupyter notebooks.
- Git and GitHub (version control).
- Ubuntu (project structure management).

## How to Use This Repository:
If you are reviewing this repository:
- Start with any section's README to understand the scope.
- Explore commands.md to see commands used to see their applications.
- Read tests.md to understand decision-making and growth.

If you are learning alongside it:
- Follow sections in order.
- Attempts tests before checking solutions.
- Focus on understanding, not speed.

This repository is actively maintained and expanded as I progress through new topics and projects.
